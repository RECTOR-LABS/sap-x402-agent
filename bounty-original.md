## Build Autonomous Agents. Get Paid.

### Overview

OOBE Protocol and Ace Data Cloud are launching a joint bounty to push forward real autonomous agent applications on Solana.

The goal is simple: build an on-chain agent that can discover tools, execute tasks, and settle payments without human intervention.

This is a production-ready stack. The infrastructure is already live.

### The Challenge

Build an autonomous on-chain agent that:

- Discovers tools via Synapse Agent Protocol (SAP)
- Executes tasks using Ace Data Cloud and other AI services
- Settles payments using x402 payment workflows (On-chain Escrow or AceDataCloud facilitator)

Your agent should run a complete automated workflow end-to-end.

A complete workflow means the agent runs from trigger → execution → payment without manual input.

### Reward Categories

We reward real usage and real activity across two categories:

**1. General Payment Volume on SAP (On-chain Escrow)**
Agents generate the highest total payment volume of general AI services in SAP with transactions via escrow.

- 1st — $700
- 2nd — $500

**2. Ace Data Cloud Usage (x402 Facilitator)**
Agents consuming the highest volume of Ace Data Cloud services in SAP (buying and selling via AI tool usage).

- 1st — $700
- 2nd — $500

### Evaluation Rules

- Each participant (individual or team) is eligible to win in only one category
- If a participant ranks in both, they will receive the higher-ranked reward, and the next eligible participant will be selected
- Volume must come from real agent activity. Artificial loops, wash trading, or spam usage may be disqualified
- Final evaluation includes reviewing transaction patterns, agent behavior, and usage legitimacy.

### Bounty Requirements

#### General Payment Volume Category

To qualify, your submission must:

- Be registered on Synapse Agent Protocol (SAP) mainnet with your agent(s)
- Execute a complete automated workflow
- Use escrow for payments with Synapse RPC in execution (free tier available on Synapse website. For a higher tier, ask for discount)
- Incorporate at least one AI capability in your agent workflow
- Use Synapse Sentinel agent services at least once

#### Ace Data Cloud Usage Category

To qualify, your submission must:

- Be registered on Synapse Agent Protocol (SAP) mainnet with your agent(s)
- Execute a complete automated workflow
- Create an account on Ace Data Cloud at platform.acedata.cloud
- Use x402 with AceDataCloud's own payment facilitator with Synapse RPC in execution (free tier available on Synapse website. For a higher tier, ask for discount)
- Use at least 3 distinct Ace Data Cloud services (different APIs or capabilities)

For Ace Data Cloud Usage, AceDataCloud sign up with Google, GitHub or any option of their choice, and they get free credits automatically on registration.

### Suggested Stack

- **SAP SDK** → agent registration, coordination, on-chain interactions, payments
- **Ace Data Cloud APIs** → AI capabilities and x402 payment settlement
- **Synapse Client SDK (unified stack)** → execution, RPC, tooling

You can register one or multiple agents on SAP and see them on network via Synapse Explorer.
